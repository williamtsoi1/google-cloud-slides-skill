# Google Cloud Slides Layout Catalog

This document provides a comprehensive list of slide layouts available in the Google Cloud 2025 presentation template. Use these layouts to structure your content appropriately.

## Title and Agenda Layouts

- **Cover / Title**: Large bold headline left, date below, Google Cloud logo bottom-left, "C" arc decoration right.
- **Contents (horizontal)**: "Contents" heading, numbered items with blue numbers right-aligned.
- **Contents (vertical)**: "Contents" heading, numbered list left-aligned.
- **Agenda (3-col grid)**: Rainbow bar, 3-column grid: time + title + description.
- **Agenda (2-item list)**: Rainbow bar right-aligned, time + bold title + description rows.

## Section Dividers

- **Section divider (white)**: Rainbow bar top, large section number + title.
- **Section divider (green)**: Full green bg (`#34A853`), section number top-left, title + subhead list right.
- **Section divider (blue)**: Full blue bg (`#4285F4`), same layout as green.
- **Section divider (red)**: Full red bg (`#EA4335`), same layout as green.

## Content Layouts

- **Title + body (1-col)**: Slide title top-left, body text below.
- **2-column text**: Slide title, two equal text columns.
- **3-column text**: Slide title, three equal text columns.
- **Bullets**: Title + bullet list (6–8 items max, no punctuation).
- **Icon + text grid**: 3-col or 4-col grid with icon + title + description per cell.
- **Table**: Standard data table with blue header row, alternating light-blue row fills.

## Data and Chart Layouts

- **Stat / Big number**: Large bold stat (e.g., "40%") in Google Blue, label below, supporting text.
- **3-stat row**: Three stats side by side with labels.
- **Bar / column chart**: Chart.js-style bar chart with Google Blue fills, labeled axes.
- **Line chart**: Line chart with Google Blue line.
- **Pie / donut chart**: Pie chart in Google brand colors.
- **Flow chart**: Org/flow chart with dark blue boxes and light blue summary row.

## Statement and Quote Layouts

- **Big statement (light)**: Full-slide large text, first phrase in Google Blue, rest in dark.
- **Big statement (dark)**: Dark bg, large white text left, rainbow bar right.
- **Big statement + rainbow bar (light)**: Rainbow bar top-left, large bold text below.
- **Staircase text**: Indented staircase lines with diagonal rainbow line left.
- **Quote (dark)**: Dark bg, large quotation marks top-center, quote text, attribution in Google Blue.
- **Two-tone headline (light)**: Very large headline, first line dark, second line Google Blue.
- **Two-tone headline (dark)**: Same as above but dark bg.
- **Split dark/light**: Left half dark bg with white text, right half white bg with dark text.

## Image Layouts

- **Image left + statement**: Image left half, large statement text right.
- **Single column with image**: Title + body text left, tall image right column.
- **Image left + headline**: Large image left, large headline text right (no footer).
- **Image right + headline**: Large image right, large headline text left (no footer).
- **Bullets with image**: Title + bullets left, tall image right.

## Maps and Closing

- **US Map**: Title top-left, US map graphic center-right.
- **World Map**: Full-slide world map with blue dot markers.
- **Thank You**: "Thank you" left, large "C" arc decoration right, Google Cloud logo bottom-left.
